## ⚙️ 기능 설명

- 로그인 / 로그아웃

- 각 게시판의 기능 ( 커뮤니티 / 사람모집 / 플리마켓 )

  - 게시글 CRUD
  - 북마크, 댓글 기능
  - 검색, 정렬, 필터링 기능

- 마이페이지
  - 사용자가 작성한 글, 댓글 단 글, 북마크 한 글 게시판별로 확인 기능
  - 사용자의 프로필이미지, 닉네임 수정 기능
