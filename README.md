## ⚙️ 기능 설명

- 홈
  - 모든 게시판 한눈에 보기 가능
- 커뮤니티
  - 게시글 CRUD 기능
  - 게시글 댓글 생성, 삭제 기능
  - 게시글 북마크 기능
  - 검색 기능
  - 필터링 기능
  - 게시글 페이지네이션
- 마이페이지
  - 작성한 글, 댓글 단 글, 북마크한 글 조회 가능
